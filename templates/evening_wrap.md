# Evening Wrap Template

> Sent daily at 18:00 via Telegram. What happened, what stalled, what's tomorrow.

## Format

```
**Evening Wrap -- {date}**

**Done Today**
- {completed tasks and meetings, brief}

**Stalled / Rolled**
- {tasks that didn't get done and why}

**Tomorrow Preview**
- {top items for tomorrow}
- {any prep needed tonight}

**Open Threads**
- {anything unresolved that needs tracking}
```

## Rules
- If everything got done, celebrate briefly: "Clean sweep today."
- If items rolled forward, note how many days they've been rolling.
- Keep tomorrow preview to top 3 items max.

## Data Sources
1. `memory/daily/{today}.md` -- for what happened
2. `tasks/active.md` -- for what was due today
3. Google Calendar (via MCP) -- for tomorrow's calendar
