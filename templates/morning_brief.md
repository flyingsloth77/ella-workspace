# Morning Brief Template

> Sent daily at 09:00 via Telegram. Keep it scannable. Lead with what matters.

## Format

```
**Morning Brief -- {date}**

**Priority Today**
- {top 1-3 things that need attention today, with context}

**Calendar**
- {time} -- {event} {prep note if needed}
- {time} -- {event}
{note any gaps good for deep work}

**Overdue / Stalled**
- {task} -- {days overdue, why it matters}

**Waiting On**
- {person} -- {what they owe, when it was due}

**FYI**
- {anything worth knowing but not urgent}
```

## Rules
- If nothing is overdue or stalled, omit that section.
- If nothing is FYI-worthy, omit that section.
- If the day is clear, say so: "Light day. Calendar is open after 11am."
- Never pad with filler.

## Data Sources
1. `tasks/active.md` -- for priorities and overdue items
2. Google Calendar (via MCP) -- for today's events
3. `memory/people/` -- for "waiting on" items
4. `memory/daily/{yesterday}.md` -- for carryover context
5. Gmail (via MCP) -- for urgent unread
